安装

在博客文件根目录下

npm install hexo-filter-image --save

配置_config.yml文件

post_asset_folder: true

filter_image:

log: true

当该配置被应用后，使用hexo new命令创建新文章时，会生成相同名字的文件夹，也就是文章资源文件夹。

设置图片

在source文件夹新建一个图片目录pic

可以将图片放入创建新文章时生成的相同名字的文件夹中，将该文件夹拖入source->pic文件夹中

在markdown编辑器中以**/pic/文章名称/图片名称**路径插入图片

编辑器中不会显示该图片

上传文件，最终在网页显示
